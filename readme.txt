== Installation ==

1. Wgraj pliki wtyczki do folderu `/wp-content/plugins/plugin-name`, lub zainstaluj go przez panel administracyjny WordPress.
2. Aktywuj wtyczkę przez w zakładce 'Wtyczki' w WordPressie
3. Przejdź do zakładki UpMenu aby skonfigurować wtyczkę
4. Uzupełnij pole wpisując adres swojej subdomeny wygenerowany w panelu UpMenu
5. W dowolnym miejscu w treści strony wklej tag [upmenu], aby wyświetlić menu na swojej stronie.
